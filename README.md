## Install and run the project
<br />
Git clone
<pre>
    git clone git@github.com:martsDev/Easy-Slider.git
</pre>

<br />

Access project directory
<pre>
    cd Easy-Slider
</pre>

<br />

Run npm install
<pre>
    npm install
</pre>

<br />

Run the project
<pre>
    npm start
</pre>
